💬 FluxAI – Frontend (React + Tailwind)

FluxAI Frontend is a modern, responsive UI for the AI-powered customer support platform.
It provides:

Real-time chat interface

Admin dashboard

FAQ & document management pages

Authentication UI

Light/dark theme

Deploys easily to Vercel.

✨ Features
🧠 AI Chat Interface

SSE-based real-time responses

Upload-based grounding

Clean chat UI

🔐 Authentication

Login for admin

Automatically stores JWT

📄 Document Management

Upload PDF/DOCX

List & delete documents

❓ FAQ Manager

Add, edit, delete FAQs

🎨 UI/UX

TailwindCSS

Mobile-friendly

Dashboard layout

🏗️ Tech Stack

React.js

Vite

TailwindCSS

Axios

React Router

Vercel (deployment)

📁 Project Structure
src/
  pages/
  utils/
  components/
  App.jsx
  main.jsx

▶️ Run Locally
npm install
npm run dev

VITE_API_URL=https://fluxai-910t.onrender.com/api
